## Grid Calc Sass

Your standard SASS grid template that allows you define your gutter width as a pixel value.

### Usage



``` html

<div class="row">
    <div class="column-2"></div>
    <div class="column-10"></div>
</div>

<div class="row">
    <div class="column-3"></div>
    <div class="column-9"></div>
</div>

<div class="row">
    <div class="column-3"></div>
    <div class="column-3"></div>
    <div class="column-3"></div>
    <div class="column-3"></div>
</div>

```
